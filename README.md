# Outfit Generator

weather based outfit generator built with react, input you're clothing pieces and it give you and outfit based on the vibe you are going for.

#How it works

-gets the users location and fetches weather data
-generates outfit based on the current temperature and users chosen occasion
- users are able to view the clothing items they have entered and filter that they want to see
- Handles missing wardrobe items — shop for it, add it, or pick an alternative
- Save your favourite outfits and view them in your Outfit Closet

  # Tech Stack
  - React + Javascript
  - Vite
  - React Router DOM
  - Open-Meteo API - weather data
  - Browser Geolocation API — auto-detects your location
  -  localStorage — persists wardrobe and saved outfits
 
#  How to run locally
- git clone https://github.com/daisyy1275/outfit-generator
- cd outfit_generator
- npm install
- npm run dev
